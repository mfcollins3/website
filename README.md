# Michael Collins's Personal Website

## About this Repository

This Git repository contains the source code and raw content for the personal website and blog of Michael Collins. I use this website to publish personal and professional information about myself including sharing technical topics that I am interested in, projects that I am working on, or general opinion articles. The information shared on my website is intended to help me learn and share my thoughts, and to share what I am learning to educate others. The source code for my website is public and accessible under an open-source license to help others with developing their own personal website and blog by seeing how I built mine.

The views expressed on this website and blog are mine alone and do not necessarily reflect the views of my employer.
