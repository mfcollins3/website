# Required Software for Development Container Development

This document lists the required software that must be installed to build and run the development container for developing the website. The development container is advantageous because it is a stable and isolated Linux container that is pre-configured with all software development tools and libraries necessary to add content to or make modifications to the website.

The following software is required to be installed for developing the website using the development container:

1. [Docker Desktop](#docker-desktop)
1. [Visual Studio Code](#visual-studio-code)
1. [Remote Development Extension Pack for Visual Studio Code](#remote-development-extension-pack-for-visual-studio-code)

## Docker Desktop

| Operating System | Required? |
| ---------------- | --------- |
| Apple macOS | :white_check_mark: |
| Linux | :white_check_mark: |
| Microsoft Windows | :white_check_mark: |

[Docker Desktop](https://www.docker.com/products/docker-desktop/) is a set of tools for building container images and running containers. Docker Desktop is intended for use by software developers and individuals who have the need to run containers locally on their machine.

Docker Desktop can be downloaded and installed from the [Docker Desktop website](https://www.docker.com/products/docker-desktop/).

## Visual Studio Code

| Operating System | Required? |
| ---------------- | --------- |
| Apple macOS | :white_check_mark: |
| Linux | :white_check_mark: |
| Microsoft Windows | :white_check_mark: |

[Visual Studio Code](https://code.visualstudio.com) is an open source text editor created by [Microsoft](https://code.visualstudio.com). Visual Studio Code has a large ecosystem of extensions that add software development tools, programming language support, and product integrations into Visual Studio Code to make it a powerful development environment.

Visual Studio Code can be downloaded and installed from the [Visual Studio Code website](https://code.visualstudio.com).

## Remote Development Extension Pack for Visual Studio Code

| Operating System | Required? |
| ---------------- | --------- |
| Apple macOS | :white_check_mark: |
| Linux | :white_check_mark: |
| Microsoft Windows | :white_check_mark: |

[Remote Development Extension Pack for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack) is a pack of Visual Studio Code extensions that support remote development scenarios using Visual Studio Code. The Remote Development extension pack includes support for using SSH to connect to remote machines or for building and running development containers in Visual Studio Code using [Docker Desktop](#docker-desktop).

The Remote Development Extension Pack can be installed from the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack) or from the Extensions tab in Visual Studio Code.
